# examenI
Servicio de correo: Un servicio de correo debe
obtener información para realizar notificaciones, por
lo tanto, se requiere almacenar una lista de
notificaciones que contenga la siguiente información:
datos de la persona que envía, mensaje del correo,
título del correo, fecha del envío del correo (debe ser
mayor al día actual), CC y CCO. Se debe seleccionar en
una lista de multiselección las personas destinos de la
notificación.
Validar que la fecha de envío sea mayor al día actual
